# F1 Race Analysis & Machine Learning Project

## Overview

This project analyzes Formula 1 race data from the 2024 Bahrain Grand Prix using FastF1, Pandas, Matplotlib, Seaborn, and Scikit-learn.

## Objectives

* Load and clean F1 race data
* Perform exploratory data analysis
* Visualize lap and sector performance
* Build a Random Forest Regression model
* Predict lap times
* Detect anomalous laps

## Technologies Used

* Python
* FastF1
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Dataset

2024 Bahrain Grand Prix Race Session loaded using FastF1.

## Visualizations

* Lap Time Distribution
* Lap Time by Tyre Compound
* Sector Comparison
* Speed vs Lap Time Correlation
* Predicted vs Actual Lap Time
* Feature Importance
* Anomaly Detection

## Machine Learning Results

* MAE: 0.3364
* RMSE: 0.4909
* R² Score: 0.9859
